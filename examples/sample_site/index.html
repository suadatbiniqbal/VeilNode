Features

Privacy First

All connections are routed through the Tor network, ensuring complete anonymity for both the host and visitors. No IP addresses are logged or exposed.

Easy Setup

No complex configuration required. Simply point VeilNode to your website directory and you're live on the dark web in seconds.

Cross-Platform

Works seamlessly on Linux, macOS, Windows, and even Termux on Android. Host from anywhere, on any device.

Technical Details

  • Built on Python's native HTTP server
  • Uses Stem library for Tor integration
  • Creates ephemeral hidden services (no persistence)
  • Supports any static web content (HTML, CSS, JS)
  • Automatic port configuration
  • Clean command-line interface
Python 3.7+ Tor Network Stem Library HTTP Server Cross-Platform

How It Works

  1. VeilNode starts a local web server on your machine
  2. Connects to your system's Tor service
  3. Registers an ephemeral hidden service
  4. Tor generates a unique .onion address
  5. Visitors access your site through Tor Browser
  6. Traffic is encrypted and routed through multiple nodes
